# 《药娘的天空》精修版
作者：乱世银娘

小说原文已被各大小说平台下架，且内容存在部分错误，影响阅读体验。本项目旨在“修复”此小说。

个人力量有限，请大家积极通过 GitHub Issues 和 Pull Requests 提出修改建议。

[项目仓库](https://github.com/yndtk/YaoNiangDeTianKong-Fix)
